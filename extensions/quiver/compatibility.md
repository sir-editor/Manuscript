# Compatibility

Quiver works both on **macOS** and **Windows**.

| application | version |
| ----------- | ------- |
| Premiere Pro | 2020 and higher |

## Upgrading from Quiver 1.1

Quiver 1.5 is a major rewrite and is **not compatible** with Quiver library projects from version 1.1 and earlier.

On first launch after upgrading, Quiver offers to import legacy library projects into the new folder-based format. See [Quiver folders](quiver-folders.md#legacy-library-import).

{% hint style="info" %}
After migration, manage all items from the Quiver main panel. The old library projects panel and "Quiver" bin in the Project panel are no longer used.
{% endhint %}

# Item settings

Right-click an item to open its context menu. Besides add method, target track, and label, Quiver 1.5 adds settings for blend mode, volume, duration, and in/out points.

<figure><img src="../../.gitbook/assets/Quiver_15_item_settings_menu.png" alt=""><figcaption></figcaption></figure>

## Method, tracks, and label

These settings work the same as in earlier versions. See [Main panel](main-panel.md#add-methods) for add methods and target track behavior.

* **Method** — Overlay, Overwrite, or Insert
* **Video track** / **Audio track** — destination track by name
* **Label** — clip label color in the timeline

## Blend mode

For video items, choose a blend mode that will be applied when the clip is added to the timeline.

## Volume

For audio items, open the **Volume** dialog to set the clip volume applied on insertion.

## Duration

For still images, open the **Duration** dialog to set how long the image stays on the timeline.

## In/out points

For video and audio items, use **Set in/out points** to trim the clip on insertion. The preview shows dark zones outside the in/out region.

Clear in/out points with **Clear in/out points** from the same submenu.

## Referenced items

If a referenced file is missing, the item appears offline. Right-click and choose **Link item** to point it to a new file location.

## Panel shortcuts

| Action | Shortcut |
| ------ | -------- |
| New folder | Shift+N |
| New item | Shift+I |
| New sequence | Shift+S |
| Edit sequence | Shift+E |
| Delete | Backspace |
| Rename | Enter |
| Duplicate | Shift+D |
| Reveal in Finder/Explorer | Shift+R |
| Set poster frame | P |
| Set start marker | M |

## Reveal commands

From a toolbar item's context menu:

* **Reveal in Quiver** — opens the main panel and selects the item
* **Reveal in Finder/Explorer** — shows the file on disk

# Quiver folders

In Quiver 1.5, library projects are replaced by **quiver folders** — top-level folders stored on disk in your Quiver data path.

Each quiver folder is a self-contained library of items. All media lives in the Quiver main panel; nothing is preloaded into Premiere projects.

## Activate a quiver folder

Click a top-level folder in the tree to make it active. The blue highlight shows which quiver folder shortcuts will add from.

You can also activate quiver folders with Spell Book commands under **Activate Quiver Folder** (e.g. **Quiver 1**, **Quiver 2**, …). [Learn more](spell-book-shortcuts.md).

## Quiver data path

By default, Quiver stores all folders and media in its data directory. You can change this location in [Settings](settings.md#quiver-data-path).

This is useful if you keep your files in the cloud and want to share quivers across multiple machines.

## Legacy library import

Quiver 1.5 is **not compatible** with Quiver library projects from earlier versions.

On first launch, Quiver will offer to import any legacy library projects it finds. Imported items are converted into the new folder structure.

{% hint style="warning" %}
Legacy import is a one-time migration. After upgrading, continue working with quiver folders in the main panel.
{% endhint %}

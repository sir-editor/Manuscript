# Compatibility

Grave Robber works both on **macOS** **(10.14.5 and higher)** and **Windows**.

| application      | version              |
| ---------------- | -------------------- |
| **Premiere Pro** | **2020, 2021, 2022** |

For Grave Robber to work one should:

* macOS: give permissions to Premiere Pro
* Windows: check antivirus

# Uninstall

To uninstall Watchtower, remove\
“**knights_of_the_editing_table.watchtower**” folder from:

| os          | path                                                        |
| ----------- | ----------------------------------------------------------- |
| **macOS**   | `/Library/Application Support/Adobe/CEP/extensions/`        |
| **Windows** | `C:/Users/*username*/AppData/Roaming/Adobe/CEP/extensions/` |

```markup

```

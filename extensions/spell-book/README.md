# Spell Book \[Excalibur version]

There are 2 kinds of Spell Book:

* old Spell Book, that comes with **Excalibur 1**
* [new Spell Book](<README (1).md>), that comes with:
  * Quiver
  * Watchtower
  * Grave Robber
  * Anchor
  * Arrow
  * Portal

New Spell Book will eventually replace old one.
